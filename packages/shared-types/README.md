# @jobhunter/shared-types

TypeScript types generated from the API's OpenAPI schema, consumed by the frontend so the
client/server contract stays in sync ([ADR-001](../../docs/architecture/decisions.md#adr-001-monorepo-with-apps-and-shared-packages)).

Generation is wired up in **PR-010** (OpenAPI typed client). Until then this package is a
placeholder.
